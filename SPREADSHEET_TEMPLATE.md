# スプレッドシートテンプレート

## 化合物データ（compounds.csv）

以下の列を含むスプレッドシートを作成してください：

| id | name | type | formula | atoms | bonds |
|----|------|------|---------|-------|-------|
| 1 | メタン | アルカン | CH4 | (JSON形式) | (JSON形式) |

### 列の説明

- **id**: 化合物の一意のID
- **name**: 化合物の名称（日本語）
- **type**: 化合物の種類（アルカン、アルケン、アルキン、アルコール、アルデヒド、ケトン、カルボン酸、エーテル、芳香族）
- **formula**: 化学式（例: CH4, C2H6）
- **atoms**: 原子データ（JSON形式、オプション）
- **bonds**: 結合データ（JSON形式、オプション）

### atoms/bondsのJSON形式

構造式データを直接管理する場合は、以下のJSON形式を使用：

**atoms例:**
```json
[{"id":"C1","element":"C","x":200,"y":200},{"id":"H1","element":"H","x":110,"y":110}]
```

**bonds例:**
```json
[{"from":"C1","to":"H1","type":"single"}]
```

**注意**: atoms/bondsが空の場合は、既存の化合物データから自動的に構造式が取得されます。

## 反応データ（reactions.csv）

以下の列を含むスプレッドシートを作成してください：

| type | from | reagent | to | description |
|------|------|---------|----|-------------|
| substitution | メタン | 水素→塩素 | クロロメタン | メタンに水素を塩素で置換すると何になる？ |
| synthesis | メタン | 塩素 | クロロメタン | メタンに塩素を反応させると何になる？ |

### 列の説明

- **type**: 反応の種類（substitution: 置換反応、synthesis: 合成反応）
- **from**: 反応前の化合物名
- **reagent**: 試薬または反応条件
- **to**: 反応後の化合物名
- **description**: 問題文

## 使用方法

1. Googleスプレッドシートで上記の形式でデータを作成
2. 「ファイル」→「ダウンロード」→「CSV（.csv、現在のシート）」でCSVファイルをエクスポート
3. エクスポートしたCSVファイルを `src/data/` フォルダに配置
4. アプリを再ビルド
